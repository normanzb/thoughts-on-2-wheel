记录骑摩托时突然冒出的各种脑洞
这是一个记录在欧亚大陆最西边的小岛上骑摩托时、站在山顶时、躺在小溪旁、坐在公园的长椅上时，大脑放空却突然冒出的各种脑洞的博客。有时候也会聊一聊计算机技术，或是欧洲值得骑车去的一些地方，分享一些粗浅的骑行经验。
